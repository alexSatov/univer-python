Game "Checkers" (International checkers, 100-cells field)
Rules: https://en.wikipedia.org/wiki/International_draughts
For developing this game was used Python (3.5 version) and graph lib PyQt5 !!!necessary to start!!!
Commands for Launching:
>main.py (if correct default python version)
>python main.py (for writing arguments)
>python main.py -h / --help (for reading info about arguments)
If something goes wrong, try to set default settings (if it's not help, reinstall game):
>python main.py -d / --default
