Game Lines
This is cross-platform game. For the graphic part used library PyQt5. 
To start the game you need to have python version >= 3.4. If you have another version of python make sure that in the console default version is 3.4 
otherwise you can't launch the game. 

Instructions for changing python version:
    In Windows: change your system variable PATH 
    In Unix: exist special version control modules

Run command:
>python lines.py

Game help: lines.py -h or lines.py --help
