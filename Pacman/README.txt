Игра PACMAN
Для написания игры использовалась графическая библиотека PyQt5 и python версии 3.5 (!!!они же необходимы для запуска!!!)
Если у вас есть другая версия python, убедитесь что по умолчанию в консоли стоит версия 3.5, иначе игра не запустится.
Изменить версию python в консоли Windows можно, лишь изменив системную переменную path, для Unix имеются специальные модули контроля версий.
Команды для запуска:
>main.py (если файлы .py по умолчанию запускаются через интерпритатор python3.5)
>python main.py (читы работают только так)
Для получения подробной информации по игре вызовите справку:
>python main.py /? - Windows
>python main.py --help - Unix
Для получения статистики вызовите таблицу рекордов:
>python main.py rec - Windows
>python main.py -r - Unix
